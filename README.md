# Gemini-Retrieval-Augmented-Generation-RAG-system-with-LLMs-from-Scratch
This project is a Retrieval-Augmented Generation (RAG) system using Google Gemini LLMs. Users can upload documents (.txt, .md, .pdf, .docx) and ask questions. The system generates answers strictly based on the uploaded content, providing reliable, context-specific responses without external knowledge.
